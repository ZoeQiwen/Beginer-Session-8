# 🧩 Python Functions Practice Project

A simple Python project focused on understanding how functions work, including parameters and basic calculations. This project demonstrates how to create reusable code and interact with users through function calls.

---

## 📌 Features

* 👋 **Greeting Function**
  Greets a user by their name.

* 💬 **Conversation Function**
  Simulates a simple conversation with a user based on their name and location.

* ⏳ **Life in Weeks Calculator**
  Calculates how many weeks you have left if you live up to 90 years.

---

## 🧠 Concepts Used

* Functions (`def`)
* Parameters & Arguments
* f-strings (formatted strings)
* Basic math operations
* Code reusability

---

## 🚀 How to Run

1. Install Python
2. Copy the code into a file (e.g., `main.py`)
3. Run the file:

